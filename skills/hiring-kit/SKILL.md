# Hiring Kit

## Overview

Takes a role and produces a hiring package: recruitment funnel with conversion targets, a job description, a sourcing channel plan, a phone-interview script, and an objection-handling guide.

## When to Use

- User is opening a vacancy and needs structure
- Few or low-quality candidates are applying
- User needs an interview script or struggles closing candidates
- Setting up a repeatable hiring process

## Core Pattern

### Step 1 — Gather input

Ask for (if not provided):
- Role, primary function, seniority
- Salary range (real — vacancies without it lose responses)
- 5–7 main tasks (not duties), must-have vs nice-to-have requirements
- Format, schedule, location, probation, benefits

### Step 2 — Build the recruitment funnel

Stages candidates pass to get an offer. Two principles: проще воронка → выше результат; для разных способов найма — своя воронка.
Conversion targets:
- До встречи с руководителем — норма ниже 50%.
- После — выше 50%.
- **Общая конверсия воронки ≥ 5%.** Ниже → ищи этап с потерей кандидатов.

### Step 3 — Job description

- Профиль должности до текста.
- Стандартное название (без креатива), указать специализацию.
- Реальная зарплатная вилка.
- 5–7 задач (не функций); требования разделить на обязательные/желательные.
- Формат, график, адрес, испытательный срок, льготы.
- Заполнить профиль работодателя (лого, фото, соцсети), отвечать на отзывы.

### Step 4 — Sourcing channels

- **Работные сайты:** hh.ru (все уровни), SuperJob и Rabota.ru (регионы), Авито Работа (линейный персонал), Хабр.Карьера / Profi.ru (IT/фриланс).
- **Профсообщества:** Telegram, Discord, форумы, митапы — изучить правила, участвовать в обсуждениях, писать конкретно (продукт, команда, задача).
- **Boolean search:** запрос вида `intitle:резюме [роль] [навык] mime:pdf -образец -шаблон -вакансия`.
- **LinkedIn** (заблокирован в РФ): через `site:linkedin.com/in + запрос`.
- Первое сообщение кандидату: конкретное наблюдение из профиля → роль/стек/задача → один простой вопрос, без давления.

### Step 5 — Phone interview

Choose the format:
- **Скрининг (7–20 мин)** — базовые критерии (опыт, зарплата, готовность к командировкам). Для массовых линейных позиций.
- **Структурированное (30–45 мин)** — поведенческие вопросы по **STAR** (Ситуация → Задача → Действие → Результат), оценка по шкале. Для экспертных/управленческих.
Не выясняй по телефону: долгосрочные планы, ценности, личное (семья, здоровье, возраст, религия). Если не подходит — скажи прямо.

### Step 6 — Handle objections

Истинное vs ложное возражение: проверить гипотетическим условием («Если бы оклад был на 20% выше — продолжили бы?»).
Алгоритм: выслушать → присоединиться → проверить истинность → ответить фактами → убедиться, что снято.
Техника **Feel–Felt–Found**: «Понимаю сомнения. Многие чувствовали то же. Но потом обнаружили, что…».
Нельзя: давить дедлайнами, уходить от неудобных вопросов, обещать негарантированное.

## Output Format

1. Recruitment funnel with conversion targets
2. Job description (ready to post)
3. Sourcing plan (channels ranked for this role)
4. Phone-interview script (screening or structured STAR)
5. Objection-handling cheat sheet

## Common Mistakes

- Don't post a vacancy without a real salary range — it kills response rate
- Don't ask STAR behavioral questions in a 10-minute screening — wrong format
- Don't ask about family/age/religion — legal risk and noise
- Don't measure only "vacancies filled" — track funnel conversion to find the leak
