# People Assessment

## Overview

Takes a role/team context and produces an assessment package: the right method choice, a Performance Review template, competency/value-fit criteria as behavioral indicators, and the legal frame (attestation vs assessment).

## When to Use

- Preparing performance reviews or bonus decisions
- Need to assess competencies or culture fit objectively
- A kadровое решение (promotion/dismissal) needs justification
- Building a talent pool (кадровый резерв)

## Core Pattern

### Step 1 — Gather input

Ask for (if not provided):
- Role/team and assessment goal (development / promotion / dismissal / pay)
- Company values (if assessing culture fit)
- Whether a legal basis is required for the decision
- Data available: results, manager input, peer feedback

### Step 2 — Attestation vs assessment (choose the track)

- **Аттестация** — формальная процедура с юридической силой; может быть основанием для увольнения (п.3 ст.81 ТК РФ). Требует положения, комиссии, критериев, документов. Обязательна по закону для: педагогов (5 лет), госслужащих (3 года), работников ОПО, авиа/транспорта, спасателей.
- **Оценка персонала** — управленческий инструмент; результаты рекомендательные, не дают юридического основания для увольнения.
Rule: аттестация — только когда обязательна по закону или нужно юридически обосновать решение. Иначе достаточно оценки.

### Step 3 — Pick a method

- **Performance Review** — раз в полгода/год, развитие + цели.
- **360 градусов** — опрос руководителя, коллег, подчинённых. Только для развития, НЕ для увольнения.
- **Ассессмент-центр** — симуляция рабочих ситуаций; самая высокая валидность (0.63). Для стратегических назначений.
- **Тестирование / анкетирование** — самооценка и выбор в ситуациях.
- **Интервью по компетенциям (STAR)** — базовый метод при найме (>80% компаний, SHL).
Не применять: астрологию, анализ почерка, клинические опросники.

### Step 4 — Performance Review template (3 stages)

1. Определи ключевые ценности компании → они задают критерии эффективности.
2. Обсуждение итогов: роли и ожидания, выполненные задачи, что понравилось/нет.
3. Обмен обратной связью + план на следующий период.
HR: система регулярных 1-на-1, дата ревью заранее, результаты в цифровом виде.
Атмосфера: не критиковать без примера, не сравнивать с другими, хвалить только за реальное.

### Step 5 — Value fit via behavioral indicators

- **Value Fit** (совпадение профиля) vs **Cultural Add** (что новый человек привнесёт).
- Оценивай поведенческими индикаторами — конкретное действие, не черта характера.
  Пример «ответственность»: не «берёт ответственность», а «сообщает руководителю о срыве сроков до дедлайна и предлагает план».
- Перевод ценности в индикатор: 3–4 рабочие ситуации → позитивный/негативный индикатор → шкала 1–3 (1 не наблюдается, 2 в стандартных, 3 устойчиво в сложных).
- Юр. ограничения: ст. 3 ТК РФ (запрет дискриминации), ст. 64 (запрет необоснованного отказа). Результат ценностного ассессмента не может быть единственным основанием для увольнения.

### Step 6 — Talent pool (опционально)

- **HiPo (High Potential)** — терпимость к неопределённости, мотивация, ответственность.
- **HiPro (High Professional)** — глубокая экспертиза, горизонтальный рост.
Управление талантами — долгосрочный проект (~2 года), оптимальная доля назначений ~30%.

## Output Format

1. Track decision: attestation vs assessment (with legal note)
2. Method selection for the goal
3. Performance Review template (3 stages) or competency/value criteria
4. Behavioral indicators with 1–3 scale (if assessing values/competencies)
5. Legal cautions specific to the case

## Common Mistakes

- Don't use 360° or value assessment as the sole ground for dismissal — illegal
- Don't run "аттестация" when a simple "оценка" is enough — heavy and risky
- Don't assess traits ("ответственный") — assess observable behavior
- Don't skip the legal frame when the goal is a kadровое решение
