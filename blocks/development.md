# Блок 9. Разработка

Для технической части: сборка инструментов помощника и интерфейсов. Навыки —
спутники из открытых источников (см. CATALOG.md).

## Спутники

- `mcp-builder` (Anthropic) — сборка MCP-серверов.
- `skill-creator` (Anthropic) — создание собственных навыков.
- `frontend-design` (Anthropic) — визуальный дизайн интерфейсов.
- `web-design-guidelines` (skills.sh) — ревью UI на соответствие гайдлайнам.
- `supergoal` (skills.sh) — автономная сборка задачи от плана до результата.
- Superpowers (obra): `brainstorming`, `systematic-debugging`, `writing-plans`,
  `test-driven-development` — рабочие процессы.
