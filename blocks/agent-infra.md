# Блок 10. Инфраструктура AI-помощника

Каналы связи и служебные навыки самого помощника. Все — спутники из открытых
источников (см. CATALOG.md).

## Спутники

- `telegram-chip` (agentos) — Telegram через пользовательский аккаунт (MTProto).
- `agent-browser` (agentos) — автоматизация браузера.
- `skill-finder` (skills.sh) — поиск навыков в реестре skills.sh.
- `llm-council` (skills.sh) — прогон решения через совет из нескольких AI.
- `codex` (Codex) — делегирование задач в Codex/ChatGPT.
