# Labops — навыки для AI-помощника руководителя

Набор готовых навыков (skills) для Claude Code, собранных под курс **AI LAB** —
о том, как руководитель малого и среднего бизнеса применяет AI в операционном
управлении: найм, мотивация, регламенты, финансы, стратегия, контент, отчётность.

Каждый навык — это инструкция, которую AI-помощник автоматически подхватывает,
когда задача совпадает с его описанием. Вы просто формулируете задачу обычными
словами, помощник сам выбирает нужный навык и выдаёт результат по проверенной
методике.

## Что внутри

- **`skills/`** — 15 авторских навыков курса. Публикуются здесь файлами, готовы к установке.
- **`blocks/`** — 10 тематических блоков курса. Для каждого блока — какой навык под какую задачу.
- **`CATALOG.md`** — полный каталог: 15 авторских навыков + проверенные навыки из открытых
  источников (со ссылками на установку, без перезаливки чужих файлов).

## Установка

Навык подхватывается Claude Code на следующей сессии. Выберите один способ:

```sh
# 1. Симлинк (рекомендуется — обновляется через git pull этого репозитория):
ln -s "$PWD/skills/<name>" ~/.claude/skills/<name>

# 2. Или копия:
cp -R skills/<name> ~/.claude/skills/<name>
```

Где `<name>` — имя папки навыка, например `hiring-kit`.
