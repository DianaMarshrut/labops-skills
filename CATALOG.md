## Часть 2. Навыки-спутники (ставятся по ссылке у первоисточника)

Эти навыки мы не перезаливаем — они опубликованы авторами и остаются под их
лицензиями. Установка — по инструкции соответствующего репозитория.

### Из public-gbrain-agentos (qwwiwi) — Apache-2.0
Репозиторий: https://github.com/qwwiwi/public-gbrain-agentos (папка `skills/`)

| Навык | Что делает |
|---|---|
| `perplexity-research` | Веб-ресёрч с источниками через Perplexity Sonar. |
| `transcript` | Расшифровки YouTube через TranscriptAPI.com. |
| `twitter` | Чтение твитов, тредов, профилей (FxTwitter + SocialData). |
| `instagram-superpower` | Аналитика Instagram через HikerAPI (топ-рилсы, watchlist). |
| `markdown-new` | Чистый Markdown с любой страницы через markdown.new. |
| `groq-voice` | Расшифровка голосовых (.ogg) через Groq Whisper. |
| `telegram-chip` | Telegram через пользовательский аккаунт (MTProto/Telethon). Только если реально нужен user-account, не для ботов. |
| `agent-browser` | Автоматизация браузера (навигация, формы, скриншоты). |
